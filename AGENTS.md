# Cafe Marketing Agent — Project Memory

## Architecture conventions

- All agents are LangGraph nodes defined in `src/graph.py`
- Only `Engagement` runs as a true agentic loop; all others are deterministic nodes
- State machine: `draft → approved → scheduled → published`
- `publish` node is unreachable unless `state["approved"] == True` — this is a hard invariant
- Every agent output includes `confidence_score: float`; below 0.7 → set `human_review_required = True`
- Tool distribution follows least-privilege: publishing tools → Publishing node only; comment/DM tools → Engagement node only
- One post = one `PostAsset` (`src/schemas.py`), threaded campaign → post → media → metrics. Publishing is **idempotent on `post_id`** (skips posts already published in state or DB) — never weaken this; it prevents double-posting
- Engagement runs a **deterministic policy gate before any reply tool fires** — the LLM's `escalate` flag is never the only check

## Never do (compliance)

- Never call any publish or DM tool unless `approved == True` in state
- Never store credentials, tokens, or PII in logs or LangGraph state
- Never auto-publish Reels with trending audio — route to `manual_publish_queue` instead (Meta API restriction)
- Never initiate a DM; only reply within the 24-hour user-initiated window
- Never retry after a `DM_WINDOW_EXPIRED` error — discard the reply

## Stack decisions

- Publishing uses the **Instagram-Login API** (`graph.instagram.com`), NOT the Facebook-Login Graph API — needs no Facebook Page, sidesteps the New Pages Experience block. Tool layer: `src/tools/meta_graph.py`. Setup + gotchas: `docs/instagram-api-setup.md`.
- Post images are generated by **Gemini "Nano Banana"** (`gemini-3.1-flash-image`, `src/tools/image_gen.py`), reference-guided by the mascot. Image models require Gemini billing — they are NOT on the free tier.
- Supabase: pgvector brand memory + a **public Storage bucket** for post images (Instagram fetches `image_url` server-side, so images must be public JPEGs). Schema in `db/schema.sql`. Skip Zep until a temporal graph is needed.
- Claude Haiku for caption drafts; Claude Sonnet for strategy and review passes.
- Meta tools call the Graph API directly (httpx); Composio remains an optional future MCP layer.
- Langfuse for tracing and eval pass-rate tracking (Phase 5+).
